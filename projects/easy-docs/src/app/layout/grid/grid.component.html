Grid

import { GridModule } from 'easy-framework';

In order for grid to work, a parent container must have a .grid class.

Because of the complexity and flexibility of grid, EASY only provides simple styles for layout.

For advanced layouts, provide your own styles. See A Complete Guide to Grid for details.

Container Columns and Rows

Use .cols-[1 - 12] and .rows-[1 - 12] classes on the grid container to build your grid.

Note: The .cols and .rows classes use the repeat(X, 1fr) function where X is the number provided to produce the grid.

Container Column and Row Gaps

Use .col-gap-[xs || sm || md || lg || xl] and .row-gap-[xs || sm || md || lg || xl] classes on the grid container to add gaps between grid items.

A gap-[xs || sm || md || lg || xl] class can be used to set both.

Grid Sticky Footer

Use .grid-sticky-footer and .rows-* classes on the main content to keep the footer at the bottom until enough content pushes it down.

Note: The .rows-* class contains the number of grid rows from the main content to the footer, inclusive.
