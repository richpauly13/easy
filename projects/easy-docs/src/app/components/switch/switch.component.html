Switch

import { SwitchModule } from 'easy-framework';

Switches are styled with a .switch-[circle || square] class on an input type="checkbox" element.

Switch labels are styled with a .switch-label class on a label element.

Note: Switches will have role="switch", a unique id, a unique for, aria-label, class=show-sr, and checked attributes added and updated automatically, if not present.
