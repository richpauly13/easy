Slider

import { SliderModule } from 'easy-framework';

Sliders are styled with a .slider-[circle || square || label] class on <input="range"> or <label> elements.

Note: If a slider does not have max, min, step, value, for, id, and/or aria-orientation attributes, they will be set and updated automatically.

Slider Colors

A slider track can be styled using a bg-* and a thumb can be styled using a text-* class from the Color page.
