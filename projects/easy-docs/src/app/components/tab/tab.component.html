Tabs are used to change views while on the same route.

Tabs are styled with .tab-btn, .tab-content, and .tabs classes.

Note: A role attribute of tab, tabpanel, and tablist, respectively, will be added automatically.

Tab Button

Tab buttons are styled with a .tab-btn class.

Note: The attributes of aria-posinset, aria-setsize, aria-controls, aria-selected, type, and a unique id will be added and updated automatically.

Note: Do not use .btn-* classes on the .tab-btn class as it will cause a component selector clash.
