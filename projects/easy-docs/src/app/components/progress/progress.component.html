Progress

import { ProgressModule } from 'easy-framework';

Progress bars are styled with a .progress class.

Note: If a progress bar does not have a value attribute, it is considered indeterminate.

Note: If a progress bar does not have a max attribute, it will be set to 100 automatically.

Progress Colors

A progress bar can be styled using a bg-* and a progress bar value can be styled using a text-* class from the Color page.

Progress Rounded

Progress bars are rounded using a .rounded class.

Note: A progress bar's value will be vertical until it reaches 100%, then it will be rounded to match the progress bar.

Progress Striped

Progress bars are striped using a .striped class.
