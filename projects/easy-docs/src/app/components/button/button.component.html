Button

Buttons are styled with a .btn[-xs || -sm || -md || -lg || -xl || -full] class.

Note: The .btn class does not have any padding, but still behaves like other .btn-* classes.

Note: The button type="button" will be automatically assigned if not present.

Group

Buttons are grouped with a .btn-group-[row || col || full] class on a parent container.

Note: The aria role="group" will be automatically assigned to the button group.

Rounded

Buttons are rounded by adding a .rounded class.

State

Buttons are disabled by adding a disabled attribute.
