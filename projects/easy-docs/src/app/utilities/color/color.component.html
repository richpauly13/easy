Background Color

Background color is styled with a .bg-[color] class.

Note: Light black is reserved for text color.

black background

blue background

dark blue background

Background Hover Color

Background hover color is styled with a .bg-hover-[color] class.

black background with a dark blue background on hover

Text Color

Text color is styled with a .text-[color] class.

Note: Light white is reserved for background color.

light black (default text color)

blue

dark blue

Text Hover Color

Text hover color is styled with a .text-hover-[color] class.

lt-black text with dark blue text on hover

Border Color

Border color is styled with a .border-[side]-[color] class.

Note: Light white is reserved for background color and light black is reserved for text color.

black border on all sides

dark blue left border

blue top border

red bottom border

green right border

purple left and right border

orange top and bottom border

Box Shadow Color

Box shadow color is styled with a .box-shadow-[1 - 5]-[color] class.

Color is optional and will default to black.

Note: Light white is reserved for background color and light black is reserved for text color.

green box shadow with a depth of 1

blue box shadow with a depth of 2

red box shadow with a depth of 3

purple box shadow with a depth of 4

black (default) box shadow with a depth of 5
