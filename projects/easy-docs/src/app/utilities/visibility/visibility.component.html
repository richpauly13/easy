Visibility

Use .hide, .show, .hidden, and .shown classes to express how an elemnt is rendered.

Adjust your browser's width to see the affect for the various classes below.

Note: Sizes refer to the viewport width xs: < 480px, sm: 480px to 768px, md: 768px to 1024px, lg: 1024px to 1366px, and xl: >= 1366px.

Accessibility

Use a .show-sr class to hide elements visually while allowing screen readers to navigate the elements.  This class is useful for checkbox and radio elements so they can be styled differently but maintain their behavior for screen readers.

Use a .show-focus class to only show elements when they are focused.  This class is useful for skip to main content elements.
