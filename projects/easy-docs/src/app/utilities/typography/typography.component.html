Typography

Many resets, normalizations, and styles have been added to some typography elements.

Headings have had the bottom margins removed and font sizes changed.

Ordered/Unordered Lists

Because <ul>s, <ol>s, and <li>s are often used for document structure, all formatting has been removed.

If you want them to represent list items, use the .list class on <ul>s and <ol>s.

  • item 2

    • sub item 2
